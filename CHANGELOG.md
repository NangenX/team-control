# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.6.1] - 2026-03-16

### Changed
- **Frontend**: 修复翻译键不匹配、迁移硬编码字符串、统一使用 t() 函数

### Translation Status
- zh: complete
- en: complete

## [1.6.0] - 2026-03-15

### Added
- Initial release with multi-agent collaboration platform
- Digital employees with persistent identity and long-term memory
- Plaza organization knowledge feed
- Multi-tenant RBAC system
- Channel integration (Slack, Discord, Feishu/Lark)
- FastAPI backend with PostgreSQL support
- React 19 frontend with TypeScript
- i18n support (zh-CN & en) via react-i18next
